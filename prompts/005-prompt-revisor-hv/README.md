# Prompt revisor de HV (PDF)

Prompt general para evaluar Hojas de Vida (HV/CV) y generar un reporte profesional en PDF.
Incluye reglas de calidad, rubrica de scoring, guia de entrevista y estandar de maquetacion.
No es un agente; es un prompt listo para pegar en el chat.

---

## Que incluye
- Roles: ATS, reclutador y lider tecnico.
- Proceso estandar de evaluacion con scoring.
- Estructura fija del reporte en PDF.
- Reglas de maquetacion y gate de calidad.
- Plantilla de mensaje del usuario.

---

## Como usar
1) Abre el archivo `prompt general.md`.
2) Completa la plantilla de mensaje del usuario.
3) Pega el prompt completo en el chat.

Archivo principal:
- `prompt general.md`
